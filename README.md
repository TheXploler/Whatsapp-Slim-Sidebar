# Whatsapp-Slim-Sidebar
A userscript to collapse the whatsapp sidebar

## Install
1. Install the [Tampermonkey](https://www.tampermonkey.net/) extension on your browser
2. Install the [script](https://github.com/TheXploler/Whatsapp-Slim-Sidebar/raw/refs/heads/main/WhatsappSlimSidebar.user.js)
3. Done!
